# Running

1. You'll need Docker and Docker Compose
2. run ```yarn```
3. run ```yarn upgrade```
3. run ```build scss```
4. run ```gulp build``` 
5. In a development environment run
```docker-compose -f docker-compose-dev.yml up```
6. open browser to http://localhost:8004


Note to self: 
3. In a production environment run
```docker-compose up -d```
